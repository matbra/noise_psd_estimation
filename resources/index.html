Resources

On this page you can download the source code that is required to generate artificial impulse disturbances and to run the impulsive disturbance classification algorithm that is described in the article.

Artificial Disturbance Generation

The MATLAB code that is the basis of the artificial disturbance generation can be found in Appendix E of Stephan Kasnitz' Master Thesis (in German). It can be downloaded here.

Impulse Restoration Algorithm

The “LSAR”, algorithm that is used to remove impulse disturbances in the audio signals is described in

Audio Restoration: An Investigation of Digital Methods for Click Removal and Hiss Reduction.

by Joseph Nuzman, published on January 26th 2004. The Octave source code has been downloaded from his website on May 17th 2014, which can be found here.

The Impulsive Disturbance Classification Algorithm

The MATLAB source code of the proposed impulse disturbance classification algorithm can be obtained from GitHub:

https://github.com/matbra/impulse-noise-classification.
